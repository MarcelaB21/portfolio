# Portafolio Personal

Este repositorio contiene el código fuente de mi portafolio web personal, desarrollado para presentar mis habilidades en desarrollo frontend, proyectos y experiencia profesional.

---

## Características del Proyecto

- Sobre mí: Desarrolladora de software fortaleciendo habilidades en desarrollo web.

- Experiencia Profesional: Timeline visual que presenta trayectoria, responsabilidades y contribuciones técnicas.

- Habilidades: Organización visual del stack tecnológico.

- Proyectos: Tarjetas con descripción y enlaces directos a proyectos.

- Diseño Responsive: Para dispositivos móviles y diferentes tamaños de pantalla.
